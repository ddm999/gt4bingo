Version

The various versions of the Bingo are designed for specific versions and mods of Gran Turismo 4 (original / Online / Randomizer / Spec II). Try to match versions whenever possible for the best experience.

To ensure that each created Bingo sheet always stays the same (even if goals are added or modified in the future) a version identifier is encoded into the URL.

Selected version:

You are not using the latest stable version. This may be intended, for example if you want to look at a linked sheet. However, if you want to generate a new Bingo sheet, consider changing version.

You are using an in-development version which may change at any time causing the same link to produce different goals. This is good if you want to make use of the latest updates to goals or for looking at a sheet someone else just linked to you, otherwise consider changing to a stable version.

How to Play

The objective is to get a "Bingo" as quickly as possible by completing the goals in five squares in a line horizontally, vertically or diagonally.

Tip: You can click on the squares to change their colour which is useful for planning or to mark off completed goals. Use the colour slider in the Options menu to choose how many colours you want to use. You can also hover over the square and press (1-6) to select a colour or (0/q) to remove the colour! Distinct symbols on the squares for each colour can be toggled in the Options menu.

Tip: Hover over a "?" to get more information about that goal.

You may want to enforce a minimum A-spec point requirement for event wins to count, to increase the challenge without affecting the duration.

When using GT4 Randomizer

If you want everyone to get the same prize cars, ensure everyone enters the same username! Otherwise, you may want to ask players to add the Bingo Seed to the end of their username, to ensure it's a random set of prize cars they've never seen before.

Playing with Others

If you're racing together with other players, generate a Bingo sheet and then link it to them so that everyone has the same goals. You can hide the table in the Options menu before generating the seed so that everyone can reveal the sheet when you start. It's usually best to start the race on the "confirm your username" popup after pressing Enter on the on-screen keyboard.

Important: Unless everyone agrees to allow it, don't cheat with the speedup function on your emulator!

Other

These rules (and goal descriptions) represent the recommended way to play Gran Turismo 4 Bingo, however if you play alone or everyone in the race agrees, then use whatever rules you want!

You can export the current sheet as JSON for use in other tools such as Bingosync.
